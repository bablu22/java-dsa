# java-dsa-journey

A structured, topic-by-topic Java repository for mastering Data Structures &
Algorithms — from fundamentals to advanced topics. Built for systematic
learning, interview preparation, and future reference.

---

## Repository Structure

```
java-dsa-journey/
├── 01-foundations/              # Java basics + OOP
├── 02-arrays-strings/           # Arrays, 2D Arrays, Strings
├── 03-core-algorithms/          # Sorting, Bit Manipulation, Complexity
├── 04-recursion-algorithms/     # Recursion, Divide & Conquer, Backtracking
├── 05-linear-data-structures/   # ArrayLists, Linked Lists, Stacks, Queues
├── 06-trees/                    # Binary Trees, BST, Heaps
├── 07-graphs/                   # Greedy, Graph Algorithms
├── 08-hashing-tries/            # Hashing, Tries
├── 09-dynamic-programming/      # DP patterns and problems
└── 10-advanced/                 # Segment Trees
```

Each topic folder contains:

- `theory.md` — Concepts, definitions, complexity analysis, key takeaways
- `revision.md` — Interview notes, common mistakes, quick revision points

## Naming Conventions

| File Type         | Convention        | Example           |
| ----------------- | ----------------- | ----------------- |
| Implementation    | `PascalCase`      | `BinaryTree.java` |
| Practice problem  | `P##_ProblemName` | `P01_TwoSum.java` |
| Algorithm class   | `PascalCase`      | `MergeSort.java`  |
| Helper/Node class | `PascalCase`      | `TreeNode.java`   |
| Interface         | `I` prefix        | `IGraph.java`     |
